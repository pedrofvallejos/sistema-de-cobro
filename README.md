# 🧮 Sistema de Gestión de Cobros y Clientes - Reparación de Electrodomésticos

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Status](https://img.shields.io/badge/Status-Completo-green?style=for-the-badge)

## 📌 Sobre el Proyecto
Este software fue desarrollado para optimizar la administración y el control de cobros en una pequeña empresa/taller de electrodomésticos. Permite llevar un registro ordenado de las cuentas, deudas y pagos de los clientes, solucionando los problemas de llevar las cuentas en papel o planillas complejas.

Como apasionado por el hardware y estudiante de desarrollo, diseñé esta herramienta pensando en la practicidad que requiere el trabajo diario en un taller técnico.

---

## 🕹️ Características Principales (Features)
* **Gestión de Clientes:** Alta, baja y modificación de datos de clientes.
* **Control de Cobros y Deudas:** Registro de saldos pendientes y actualización automática con cada pago parcial o total.
* **Base de Datos Local:** Almacenamiento seguro de la información sin necesidad de conexión a internet.
* **Interfaz de Usuario:** Diseñada para ser intuitiva y rápida de usar entre reparación y reparación.
* **Portable (.exe):** Compilado para ejecutarse en Windows sin necesidad de instalar Python ni librerías externas.

---

## 🛠️ Tecnologías Utilizadas
* **Python:** Lenguaje principal para la lógica del sistema.
* **SQLite / File handling:** Para el almacenamiento persistente de los datos (acá poné la que usaste).
* **PyInstaller:** Utilizado para empaquetar el proyecto en un archivo ejecutable independiente.

---

## 🚀 Instalación y Uso
Al estar compilado en un archivo `.exe`, no requiere instalación de entornos de desarrollo:
1. Descargá el archivo ejecutable desde la sección de [Releases] (o cloná este repositorio).
2. Ejecutá el archivo `sistema.exe`.
3. ¡Listo! Ya podés empezar a cargar tus clientes.
